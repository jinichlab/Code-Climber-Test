◬ The Climb · Module 3

Cheminformatics & Graph Neural Networks

You're climbing a 12-hold route through the foundational skills of cheminformatics — parsing molecules, computing descriptors, building fingerprints, measuring chemical similarity, and ending with the prep work for a graph neural network that learns from molecular structure.

How it works. Each hold is a small code challenge. Get it right and your climber moves up. Get it wrong three times in a row and they fall back to base camp — so think before you send it.

The route:

  1. 1–3  Base camp · parsing SMILES strings browser
  2. 4–5  Real RDKit · molecules & molecular weight colab
  3. 6–7  Morgan fingerprints & Tanimoto similarity colab
  4. 8    Pandas + RDKit pipelines colab
  5. 9–10 Train/test splits & regression metrics colab
  6. 11–12 Graph prep & pooling for GNNs (PyTorch) colab

Holds 4–12 run on a real Python kernel — connect your free Google Colab when prompted (the colab pill in the header).
